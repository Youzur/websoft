S02

Yes, I have previous worked with HTML, CSS, and JS. I started learning these programs when I was in middle-school and then studied extensively on it in high-school.

The roles of HTML, CSS, and JS are:

    HTML
  • It is the mesh/skeletal structure of the webpage, were we can place texts and/or arrange different containers for different topics.
    CSS
  • It is the "skin" of the webpage. Here we design how our webpages can look, for example, adding a background color of red to the webpage, or increase the font size of our text.
    JS
  • It is the interactiveness of the webpage. Here we can add data to our webpage structure, for example, having a button that, when clicked, shall append data to a container in the webpage.

Basically, a browser communicates with a web server through HTTP protocol. The browser makes a request to a web server, to which the server responses back data.

The TIL of this course is to learn how to develop Web based Applications, and how different software communicate with each other to create interactive and colorful web pages.
